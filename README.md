# Sistema de Gestión de Cine

## Descripción

Aplicación de consola desarrollada bajo el paradigma de Programación Orientada a Objetos que permite administrar las salas de un cine, gestionar películas, vender entradas, registrar combos de comida y consultar el consumo total de cada cliente.

## Funcionalidades

### Agregar película
Permite registrar películas en el catálogo del cine indicando su nombre y precio.

### Crear sala
Permite crear salas asociadas a una película, definiendo la cantidad de filas y columnas de la silletería.

### Vender entrada
Permite seleccionar una sala, elegir una silla disponible y registrar el nombre del cliente.

### Consultar disponibilidad
Muestra la cantidad de asientos ocupados y disponibles de cada sala.

### Agregar combo
Permite registrar nuevos combos de comida indicando nombre y precio.

### Pedir combo
Permite asociar uno o varios combos a un cliente que tenga una silla ocupada.

### Sacar total por persona
Muestra el detalle de la compra realizada por un cliente, incluyendo el valor de la entrada, los combos adquiridos y el total a pagar.

## Conceptos Aplicados

- Programación Orientada a Objetos
- Encapsulamiento
- Composición
- Arreglos bidimensionales
- Colecciones dinámicas
- Gestión de objetos
